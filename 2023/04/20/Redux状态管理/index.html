React-redux

Redux-Toolkit（RTK）

Redux-Toolkit

RTK 在实际项目开发中，使用起来更加得心应手，它的优点在于：

  1. 可以自动和 Redux devTools 结合，不再需要依赖 redux-devtools-extension 模块来实现。
  2. 数据不再通过返回值进行修改，可以像 vue 一样直接修改。
  3. 内置了 redux-thunk 这个中间件，来实现异步操作数据。
  4. 代码风格好，采用选项式编写程序。

安装

全局引入和 react-redux 一样

基本使用

创建store

从@reduxjs/toolkit包中导入configureStore，来创建一个 store，传入一个配置对象作为参数,如果有多个 reducer 使用 reducer 对象包裹。

创建模块

在每个模块中的reducers对象中定义了多个修改状态的函数，会自动同步到模块的 action 对象中，通过导出修改函数，让组件可以同步修改共享数据。

对象结构是{type:'name/函数名',playload:函数的参数}
