构建镜像

docker build命令用于根据 Dockerfile 的指令创建 Docker 镜像。这个过程包括读取 Dockerfile 中的指令，然后按顺序执行这些指令来构建最终的镜像。每一个指令都可能创建一个新的层，这些层一起组成了整个镜像

  • -t 为构建的镜像指定一个名字和标签，通常的格式为容器名:标签
  • . 这个不能少，代表当前项目的根目录
