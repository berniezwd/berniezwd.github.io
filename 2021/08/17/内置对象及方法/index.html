内置对象及方法（常用）

发表于2021-08-17更新于2023-05-31
前端开发

内置对象及方法（常用）

知瑾2021-08-172023-05-31

数组

split

将字符串以指定字符分割成数组

join

将数组项以指定字符转换成字符串

末尾删除&末尾添加

添加：push()，返回数组的新长度

删除：pop()，返回删除项

起始删除&起始添加

添加：unshift()，返回数组的长度

删除：shift()，返回删除项

splice

splice（从哪里开始删除,总共删除多少个,使用哪些新元素进行替换）用于插入、删除、替换数组的元素

slice（开始位置，结束位置）截取数组部分,返回新数组

includes

includes()判断一个数组是否包含一个指定的值,返回布尔值

every

every（）测试一个数组内部所有元素是否都能通过条件,返回布尔值

some

some（）测试数组中至少有一个元素能通过条件,返回布尔值

find

find（）返回数组中满足提供的函数的第一个元素的值,否则返回 undefined

findIndex

findIndex（）找到第一个满足函数元素并返回那个元素的索引，若没有返回-1

filter

filter 遍历数组,把数组中的满足条件的元素筛选出来,返回新数组

map

map 遍历数组，把某个结果作为返回的数组中的某一个,返回新数组

reduce

reduce（）数组中的每个元素执行一个由您提供的 reducer 函数(升序执行),将其结果汇总为单个返回值

利用 reduce 累加数组对象的值

将二维数组转换为一维数组

计算数组中每个元素出现的次数

flat

数组降维操作，将多个嵌套数组进行扁平化

扁平化嵌套数组
通过Infinity扁平化任意深度
扁平化空项

flatMap

使用映射函数映射每个元素,然后将其展平为新数组，等效于arr.map().flat()

将对象数组中的某个字段，扁平化处理后形成新的数组

字符串

indexOf

查找子字符串，返回第一次出现指定值的索引，未找到则返回-1

空格也可以当做子字符串来截取

slice

截取字符串中的值，slice(起始位置，结束位置<可选>)，若没有传入结束位置，会返回剩余字符串

转换大小写

转大写：toUpperCase()

转小写：toLowerCase()

replace

替换字符串的某部分

includes

判断一个字符串是否包含在另一个字符串中 返回布尔值

endWith

判断当前字符串是否以另外一个给定的子字符串结尾的

Math

获取范围内随机数字[0,1)

数字四舍五入

数字向上取整

向下取整

绝对值
