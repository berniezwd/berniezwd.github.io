《JavaScript语言精粹一》从零单排之 语法/对象篇

发表于2022-11-23|更新于2022-11-30|Vue

空白

JavaScript提供两种注释形式，块注释/* */ 和行注释 / /。

注释一定要精确描述代码。没有用的注释比没有注释更糟糕。

数据类型

JavaScript的简单数据类型包括：

  • 数字
  • 字符串
  • 布尔值
  • null
  • undefined

对象类型包括：

  • 数组
  • 对象
  • 函数
  • 正则表达式

false值

在JavaScript中，以下值表示的布尔值为false。

  • null
  • undefined
  • false
  • 空字符串’’
  • NaN
  • 数字 0

标识符

标识符被用于语句、变量、参数、属性名、运算符和标记。

值的注意的是：

  1. 一些JavaScript的关键字不可以用作命名，不允许使用保留字作为对象字面量的属性名。
  2. 标识符中带有-是不允许的，但是使用_时完全没问题（’first-name’ 和 ‘first_name’）。

数字

  • 在JavaScript只有一个数字类型。在内部被表示为64位的浮点数，和Java的dobule数字类型一样。

  • 和其他编程语言不同的是：JavaScript没有分离出整数类型，所以1和1.0的值相同。

  • 负数使用- + 数字构成，例如-15。

  • NaN是一个数值，表示一个不能产生正常结果的运算结果。NaN不等于任何值（包括它自己），可以使用isNaN(number)来检测NaN。

字符串

  • 字符串字面量可以被包在一对单引号或者双引号中，包含0个或者多个字符，所有的字符都是16位的。
  • \代表转义字符，用来把正常情况不被允许的字符插入到字符串中，例如反斜线、引号和控制字符。
  • 字符串具有length属性，"bernie".length是6。
  • 字符串一旦被创建，就无法改变它。'c'+'a'+'t' === 'cat'是 true。

语句

  • 每个<script>标签提供一组可执行的语句。

  • 当var语句被用在函数内部时，它定义的是这个函数的私有变量(不过他会变量提升至外部作用域)。

  • 语句按照从上到下的顺序被执行，可以通过条件语句(if和switch)、循环语句(while/for/do)、强制跳转语句(break/return/throw)和函数调用来改变执行顺序。

条件语句

if语句

当条件为true执行，否则执行false代码块

switch语句

  • 执行一个多路分支，它把表达式的值和所有指定的case条件进行强匹配（精准匹配），当匹配上条件时候，会执行对应的case语句，没有匹配条件的时候，执行default语句

  • case表达式不一定必须是常量，要防止继续执行下一个case，可以使用break退出switch语句。

循环语句

for 循环语句包含3个从句：初始化从句、条件从句、增量从句

for in 循环语句会枚举一个对象的所有键名。

额外补充：检测某个属性是否存在于某个对象，使用object.hasOwnProperty(属性)，返回布尔值。

for of 循环语句一般是用在数组的遍历。

while语句执行一个简单的循环，如果表达式值为假，就终止循环，为真时则循环执行代码块。（工作中似乎没用到过）

do…while语句会先执行一次代码块，再进行条件判断来决定是否继续循环执行。（工作中似乎没用到过）

强制跳转语句

try catch语句中，try执行一个代码块，并捕获该代码块 使用throw抛出的任何异常。catch从句用来接收抛出的异常对象。

  • 如果throw在try代码块中，控制流会跳转到catch从句中。

  • 如果throw在函数中，则该函数调用被放弃，控制流跳转到调用该函数的try语句的catch从句中。

  • throw通常是一个对象字面量，包含了name和message属性，通过异常对象中的这两个属性去决定继续执行后面的代码。

return语句会导致从函数中提前返回，函数默认的返回值是undefined，也可以自定义返回值。

break语句会退出一个循环语句或者switch语句

额外补充：

  • 如何退出forEach循环？（工作中有遇到过）使用try catch
  • 如何跳出本次循环？（工作中可能会遇到）使用return

表达式

  • 最简单的表达式是字面量值(比如字符串或数字)
  • 变量
  • 内置的值(true/false/null/undefined/NaN/Infinity等)
  • 以new开头的调用表达式
  • 以delete开头的属性提取表达式
  • 包在圆括号中的表达式
  • 以一个前置运算符作为前导的表达式
  • 表达式后面跟着：中置运算符与另一个表达式、三元运算符、函数调用、属性提取表达式

对象

对象是按指定规格创建新对象的表示法。

对象是属性的容器，每个属性都包括键名键值。

JavaScript里的对象是无类型的。属性名可以是标识符或字符串，属性的值是表达式，适用于汇集和管理数据。

JavaScript包含一种原型链的特性，允许对象继承另一个对象的属性，正确的使用这个特性可以减少对象初始化消耗的时间和内存。

对象字面量

使用对象字面量创建新对象的表示法，使用{}包围多个名:值，对象字面量允许出现在任何允许表达式出现的地方。

对象检索

要检索对象里包含的值，可以使用[键名]的方式。

如果键名是一个合法的字符串字面量，也可以使用.键名的方式，优先考虑.键名表示法，可读性更好。

如果对象中检索不到属性，则返回undefined

对象检索可以使用||来填充默认值

为了避免从undefined中检索错误，可以使用&&来避免错误

对象更新

对象的值可以通过赋值语句来更新，若有这个键名，则更新键值，若没有检索到键名，则会往对象中扩充这个键值对。

原型

每个对象都连接到一个原型对象，并且它可以从中继承属性。

所有通过对象字面量创建的对象都连接到Object.prototype。

原型连接在更新时是不起作用的，对某个对象做出改变时，不会触发该对象的原型。

每当创建一个新对象时，会创建一个使用原对象作为其原型的新对象。

原型连接只有在对象检索的时候才被用到，完整的检索委托过程是：

  1. 先从对象属性中检索，如果没有检索成功，Js会尝试从原型对象中检索。
  2. 如果原型对象也没有这个属性，Js会尝试从原型中检索。
  3. 最终才会检索到Object.prototype，如果完全不存在原型链中，那么结果就返回undefined
